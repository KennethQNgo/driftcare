# Favicon Assets for drift.care

This directory should contain the following favicon files:

## Required Files

1. **favicon.ico** - Multi-size ICO file (16x16, 32x32, 48x48)
2. **favicon-32.png** - 32×32 PNG
3. **favicon-16.png** - 16×16 PNG  
4. **apple-touch-icon.png** - 180×180 PNG (for iOS "Add to Home Screen")

## Design Guidelines

- **Background**: #08080c (near-black)
- **Foreground**: #d4c5a0 (champagne/accent)
- **Style**: Minimal, modern
- **Recommended mark**: Lowercase "d" with Inter font (300 weight) or simple geometric shape

## Tools to Create

- Use tools like [Favicon Generator](https://realfavicongenerator.net/)
- Or design in Figma/Sketch and export at proper sizes
- Ensure crisp rendering at small sizes
